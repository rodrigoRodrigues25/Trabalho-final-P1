# Sistema de Monitorização de um Centro de Dados

Projeto desenvolvido para a unidade curricular de Programação I do 1º ano de Engenharia de Redes e Sistemas de Computadores (IPVC).

## Ferramentas e Bibliotecas Utilizadas
* **Linguagem:** C (Norma C11)
* **Compilador:** GCC (MinGW para Windows)
* **Ambiente de Desenvolvimento:** Code::Blocks / VS Code
* **Bibliotecas Standard:** `<stdio.h>`, `<stdlib.h>`, `<string.h>`, `<time.h>`

## Estruturas de Dados Implementadas
* **Módulo 2 (Lista Ligada):** Armazenamento dinâmico das leituras dos sensores em tempo real.
* **Módulo 3 (Fila FIFO):** Gestão e atendimento de alertas gerados por anomalias nos sensores.
* **Módulo 4 (Pilha LIFO):** Histórico de controlo das sincronizações efetuadas pelo operador.
